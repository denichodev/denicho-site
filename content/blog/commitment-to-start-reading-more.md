I've always wondered how people can write something so engaging. Last week, I read this lovely [piece](https://mariozechner.at/posts/2026-03-25-thoughts-on-slowing-the-fuck-down/) and not only that I fully agree with his view on AI, but I feel really engrossed reading it. I really want to be able to write like this, to write like a human being.

I don't really read a lot of books. I read some mainstream ones, and I loved Orwell's 1984. Immediately after I brought myself to watch the movie (that is released in 1984). Same goes for Murakami's Norwegian Wood, although I don't like it as much as the former.

I'm not sure if this will help me to write better, but at least this piece will serve me as a reminder of what I'm planning to commit on this day. Let's start by going to the nearest bookstore I guess?
